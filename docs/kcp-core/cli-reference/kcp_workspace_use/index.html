kcp workspace use

Uses the given workspace as the current workspace. Using - means previous workspace, .. means parent workspace, . mean current, ~ means home workspace
kcp workspace use <workspace>|..|.|-|~|<root:absolute:workspace> [flags]

Options

  -h, --help                help for use
      --kubeconfig string   path to the kubeconfig file
      --short               Print only the name of the workspace, e.g. for integration into the shell prompt
